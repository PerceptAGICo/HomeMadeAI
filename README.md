# HomeMadeAI
This is going to be a simple model and guide to show people curious about starting in AI how to begin and kind of what's going on. For the purposes of this model's use, this is a completely copyleft style program, intended for teaching. You can learn from it, but if you claim it in order to stop ppl from using it to learn, there are going to be some problems between you and I.

## Questions Before Starting
When considering whether or not to build your own AI, there are some serious questions to take into account.

1) What is the purpose of your AI?
2) Are you doing it all online, localized, or as a hybrid?
3) Are you planning to create your own training sets or borrow some from online?
4) How are you structuring your system to operate?
5) How does your system interact with existing hardware?

These questions are crucial and important to your build.
